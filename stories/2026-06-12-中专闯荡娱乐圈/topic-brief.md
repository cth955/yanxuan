### 改造原则

```
原句：靠着大佬闯荡 → 读者以为抱大腿爽文
盐选版：靠着大佬闯荡 → 其实被当棋子，主角反杀/反利用
```

**核心升级**：「大佬」不是金手指，是**最大的悬念源**——他为什么选中专的我？

## 方案 A（推荐）：替身合约 —— 悬疑 + 身份错位

### 标题

**我中专毕业在横店跑龙套，顶流经纪人的大佬塞给我一份年薪三百万的合同——条件只有一个：不许红。**

### 导语（三有一反）

```
中专毕业，我在横店跑龙套，一天八十。
直到某个大佬递来合同：年薪三百万，条件只有一个——不许红。
我签了。
因为我长得，像那个已经「消失」了三年的顶流女星。
```

### 一句话概念

有悬念 ✅（为什么不许红？）｜有反常 ✅（三百万雇人不要红）｜有画面 ✅（横店群演 + 保姆车 + 合同）

### 类型

生活悬疑 + 现代逆袭（职场/娱乐圈）

### 主角人设

- **姓名建议**：林早（谐音「林招」，暗示被「招」进局）
- **背景**：中专化妆专业，来横店两年，住八人间，给剧组递水、跟妆
- **缺陷**：贪钱但不算蠢，嘴硬，爱在心里骂街
- **说话风格**：「我图什么？图他帅？图他有钱？我图那三百万，有错吗？」
- **能动性**：表面配合替身，暗中拍照、记时间线、查「消失女星」旧闻

### 核心冲突

大佬雇她当顶流替身在公开场合露面，掩盖真身失踪/出事；她以为自己攀上高枝，实则可能是替罪羊。

### 情绪过山车（简版）

| 阶段 | 情绪 | 事件 |
|------|------|------|
| 1-2章 | 惊+疑 | 被选中→签天价合同→第一次替身出场 |
| 3章 | 憋屈 | 全网骂「中专女攀高枝」→ 大佬不许解释 |
| 付费墙 | 抓心 | 她在替身妆里发现——真身可能不是失踪，是**死** |
| 4-7章 | 怒+布局 | 查线索→差点暴露→拿到关键证据 |
| 8-10章 | A级反转 | 「消失女星」没死 / 死的是别人 / 大佬才是凶手 |
| 11-12章 | 爽+余韵 | 直播/发布会反杀，合同变呈堂证供 |

### 付费墙断点（第3章末）

```
卸妆时，我在假发网里摸到一个东西。
一张叠成指甲盖大小的纸条。
上面是「消失女星」的笔迹，日期是——昨天。
```

### A级反转（建议）

大佬选她不是因为长得像，而是因为**真身当年也是中专毕业、也是替身出身**——整个圈子是一条吃人的流水线，她是被故意养来「接班」的下一颗棋子。

### 差异化

同类娱乐圈文多写「被包养/被宠」，本篇写「被雇佣不许红」——反常识合同驱动悬疑。

### 300字梗概

林早，中专化妆专业，在横店跑了两年龙套。某夜收工，顶流经纪人周姐把她领进保姆车，身后跟着圈内资方「大佬」沈砚。沈砚递来一份年薪三百万的合同：她需在各种场合代替已消失三年的顶流女星苏晚露面，条件只有一个——不许真正走红。全网嘲讽林早「中专女攀高枝」，沈砚却禁止她任何澄清。林早一边当替身，一边暗中调查苏晚失踪真相。她在替身道具里发现苏晚「昨天」写下的纸条，意识到苏晚可能并未失踪，而是被囚禁或已遇害。随着调查深入，林早发现自己与苏晚不仅容貌相似，连中专母校、入学年份都完全一致——她并非偶然被选中，而是被精准匹配的「下一任替身」。最终在一场无法剪辑的直播事故中，林早当众撕毁保密协议，用合同与监控证据将沈砚及背后资本链条送上热搜。故事结尾，林早没有成为新顶流，但以证人身份让「不许红」的合同Industry-wide曝光——她靠的不是大佬，是自己留的证据。

## 方案 B：选秀黑幕反杀 —— 职场逆袭 + 信息差

### 标题

**选秀被淘汰那晚，我收到封杀级资方大佬的私信：「来803，带你进圈。」**

### 导语

```
中专毕业，我参加选秀，第一轮就被刷。
后台监控却显示：我的票数被人改了。
淘汰那晚，手机弹出私信——
发信人，是业内人人喊打的封杀大佬。
```

### 一句话概念

有悬念 ✅｜有反常 ✅（封杀大佬为何找被淘汰者）｜有画面 ✅

### 类型

现代逆袭 + 悬疑

### 主角人设

- **姓名建议**：唐梨
- **背景**：中专声乐，选秀陪跑，被节目组当「学历低好欺负」的剪辑素材
- **缺陷**：看起来软，实际从淘汰前就在**秘密录音**
- **能动性**：⭐ 最强——整篇是「钓鱼执法」，大佬是猎物不是金主

### 核心冲突

她表面「靠大佬进圈」，实际是与调查记者/律师合作，用大佬的_offer_钓出选秀造假链条。

### 付费墙断点

```
803的门开了。
大佬坐在沙发上，旁边放着一份签约文件。
我认出了那份文件的公章——
和篡改我票数的那家公司，是同一个。
```

### A级反转

所谓「大佬带她进圈」，是她设计的局；大佬以为在收编棋子，其实自己才是被中专生套话的对象。

### 差异化

「靠着大佬」是假叙事——读者以为抱大腿，实则是**扮猪吃虎**。

## 方案 C：通告群陷阱 —— 生活悬疑 + 合同惊悚

### 标题

**我在通告群接了个日结两千的活，到了现场才发现，客户是圈内最大的大佬——而他要我假扮他女儿。**

### 导语

```
中专毕业，我在各种通告群抢活，妆造、礼仪、临演什么都接。
日结两千的单子，我秒接。
到了酒店我才看见甲方名字——
那个传说中一手遮天的娱乐圈大佬。
他只说了一句话：「叫我爸。」
```

### 类型

生活悬疑 + 婚姻家庭外壳（假父女）

### 核心冲突

假扮女儿参加家宴/谈判，逐步发现「女儿」身份与某起旧案、股权争夺、私生子女有关。

### 付费墙断点

```
宴席上，大佬的前妻盯着我的脸，手里的酒杯停在半空。
她问：「你叫什么名字？」
我还没开口。
大佬替我答了——
那个名字，是我死去三年的室友。
```

### 差异化

从「通告群抢活」切入，极度接地气；「假扮女儿」比「假扮明星」少见。
