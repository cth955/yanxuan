# 06 题材与选题策略

## 黄金公式（数据验证）

```
生活悬疑 + 女性视角 + 接地气场景 = 最吃推荐
```

## 题材优先级（2025-2026）

| 优先级 | 题材 | 特征 | 代表方向 |
|--------|------|------|----------|
| 🔥 1 | **古代言情/重生** | 盐选最大流量池 | 重生反杀、追夫火葬场、宫斗逆袭 |
| 🔥 2 | **现代逆袭爽文** | 过稿最快之一 | 被欺负→翻盘打脸 |
| 🔥 3 | **生活悬疑** | 验证有效 | 日常场景 + 反常发现 + 层层剥开 |
| 🔥 4 | **规则怪谈** | 2025-2026 上升 | 规则清单 + 违规触发 + 最后解释 |
| 5 | 死人文学/追悔文 | 虐中带爽 | 主角将死/已死，众人追悔 |
| 6 | 婚恋情感 | 必须带悬疑/反转 | **不能写纯催泪** |
| 7 | 底层叙事/成长逆袭 | 难度高，天花板高 | 小人物翻盘 |
| 8 | 职场逆袭 | 稳定赛道 | PUA 复仇、扮猪吃虎 |
| 9 | 悬疑推理 | 需要硬技巧 | 不可靠叙述者、多层真相 |

## 读者画像驱动的选题

### 高共鸣场景（女性向）

- 租房/搬家/合租/二手房
- 婚姻/出轨/婆婆/彩礼
- 职场 PUA/被裁/副业
- 重组家庭/继母（**非纯催泪**）
- 原生家庭/重男轻女/扶弟魔

### 验证有效的标题句式

```
我在 XX 发现了 XX
搬进 XX 的第 X 天
XX 年后，我才得知
```

## 题材硬红线 🚫

| 禁止 | 原因 |
|------|------|
| **纯亲情催泪** | 「我妈去世后找到存折」类 —— 推荐流跑不动 |
| **父母为核心情感** | 父母去世/生病/遗物/遗书/存折 |
| **以亲属关系为核心的催泪** | 「我爸再婚继母对我好」类 |
| 纯恐怖/灵异（无生活壳） | 受众窄，弱于生活悬疑 |
| 男频视角/龙傲天 | 90%+ 女性用户 |
| 脱离现实的纯脑洞 | 缺乏共鸣 |
| 政治敏感/纯色情/宣扬暴力迷信 | 官方拒稿 |

**如需家庭背景**：父母只能是配角/背景板，不能是情感核心。

## 反同质化铁律

> 连续 3 篇不得使用同一个故事模式。

每篇必须在以下维度有明确差异：
- 类型（悬疑 vs 言情 vs 职场）
- 基调（燃 vs 虐 vs 甜）
- 主角能动性（布局型 vs 发现型 vs 反击型）

**禁止重复模式**：
```
发现物件 → 追查 → 揭秘 → 催泪  （已过度使用）
```

## 追妻火葬场（言情主流套路）

### 核心机制

```
男人抛弃/忽视 → 发现女主好 → 回头追 → 女主已不爱/另有所爱 → 男的难受
```

**读者爽点**：让负心男难受，不是让女主回头。

### 变体要素

- 白月光/替身/误会
- 男二真心 vs 男主追悔
- 身份反转（女主其实更强）

## 差异化策略

### 当同类过多时

1. **换视角** —— 反派第一人称、旁观者视角
2. **换设定** —— 职业细节、地域特色
3. **换钩子** —— 同一题材用不同开篇炸弹
4. **换结构** —— 非线性叙事、不可靠叙述者
5. **换基调** —— 同一故事骨架，悬疑壳 vs 言情壳

### 选题检查清单

```
□ 一句话能否同时满足：悬念 + 反常 + 画面？
□ 是否有悬疑外壳（即使写情感）？
□ 盐选上同类是否已饱和？差异化在哪？
□ 是否触碰题材红线？
□ 与最近 2 篇是否模式重复？
□ 场景是否贴近女性生活？
□ 是否有明确爽点（非纯感动）？
```

## 热门问题投稿策略

故事发为**热门问题的回答**比发专栏流量大 10x+。

### 选题标准

| 指标 | 建议 |
|------|------|
| 关注者 | ≥ 1000 |
| 浏览量 | ≥ 100 万 |
| 已有高赞回答 | 说明有需求 |
| 调性匹配 | 继母故事 → 继母问题 |

### 类型-关键词对照

| 故事类型 | 搜索关键词 |
|----------|-----------|
| 继母/家庭 | 继母、后妈、重组家庭 |
| 婚姻悬疑 | 老公秘密、婚姻真相、出轨 |
| 职场 | 职场 PUA、被辞退、领导 |
| 租房/搬家 | 租房经历、搬家、合租 |

⚠️ 若问题下全是真实经历分享，虚构故事要写得足够「像真的」。
