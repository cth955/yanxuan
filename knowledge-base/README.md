# 知乎盐选短文写作知识库

> 面向「写小短文智能体」的结构化知识库。核心目标：帮助生成**高点击、高完读、高付费转化**的盐选风格短篇。

## 知识库定位

盐选短文不是纯文学，也不是传统网文。它的本质是：

**问答/推荐流场景下的「高密度情绪快餐」** —— 用极短篇幅完成：钩住 → 憋屈 → 反转 → 爽感 → 余韵。

平台真正考核的不是文笔多华丽，而是：**点击率、完读率、付费转化率**。

## 智能体使用建议

1. **写之前**：读 01 + 06 确定题材与读者；读 03 设计标题/导语/开篇
2. **写大纲**：读 02 + 04 + 07 做情绪过山车与卡点设计
3. **写正文**：读 05 控制文风；每600-800字对照 04 检查爽点
4. **写之后**：读 08 + 10 做终检

## 核心公式（速查）

```
爆款 ≈ 第一人称 + 极端冲突开篇 + 生活化场景 + 悬疑外壳
       + 每600-800字一个转折 + 憋屈后强反弹 + 付费墙超级悬念
       + 口语化分镜叙事 + 多层反转 + 干脆收尾
```
