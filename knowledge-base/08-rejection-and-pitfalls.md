# 08 拒稿原因与避坑

## 官方拒稿原因对照表

| 拒稿原因 | 具体表现 | 自检方法 |
|----------|----------|----------|
| 主题不合规 | 敏感、纯色情、暴力迷信 | 题材红线检查 |
| **主线不够清晰** | 读者不知道在讲什么 | 能否一句话说清主线？ |
| 文笔流畅度不足 | 读着别扭 | 大声朗读 |
| **风格与盐选相差大** | 太文学/太网文/太口水 | 对比盐选热门文风 |
| **人物不出彩** | 扁平、无缺陷 | 主角有无具体困境和缺陷？ |
| **叙事节奏慢** | 前 4000 字无吸引力 | 前 4000 字 ≥ 3 个钩子？ |
| **叙事平** | 无高潮、无 A 级反转 | 有无多层反转？ |
| 同类内容多 | 缺乏差异化 | 与已有盐选对比 |
| 重复投稿 | 一稿多投同一渠道 | 投稿记录 |
| 字数不足 | < 1 万字 | 全文 ≥ 12000 字 |

## 新手十大误区

### 1. 把盐选当「知乎精英写作」

❌ 写人性洞察、社会批判、文学性散文  
✅ 写强冲突、强情绪、强反转的通俗故事

### 2. 慢热铺垫

❌ 「我是一个普通大学生，直到那天……」  
✅ 第一句就炸

### 3. 不敢虐主

❌ 主角一直顺风  
✅ 先踩到泥里，再弹起来

### 4. 纯催泪无悬念

❌ 感动 → 泪 → 结束  
✅ 惊 → 疑 → 憋屈 → 爽 → 余韵

### 5. 段落太长

❌ 4-5 句一段的文学段落  
✅ 1-3 句，30% 独句段

### 6. 对话太少

❌ 大段叙述，对话 20%  
✅ 对话 40-50%

### 7. 付费墙后继续铺垫

❌ 第 4 章还在介绍新人物/新背景  
✅ 第 4 章立刻爆爽点

### 8. 结尾升华说教

❌ 「这件事让我明白了人生的意义……」  
✅ 故事讲完就停

### 9. AI 味太重

❌ 「然而」「不禁」「一种说不清道不明的感觉」  
✅ 口语化、不均匀段落、生活闲笔

### 10. 灵感驱动无大纲

❌ 有个点子就写，写到一半卡死  
✅ 先画情绪过山车，再填内容

## 平台用户常见差评（反面教材）

来自真实用户反馈：

- 「只有开头好看，后面越来越崩」
- 「付费后还在铺垫，觉得不值」
- 「反转太硬，逻辑不通」
- 「人物说崩就崩，前后矛盾」
- 「虐主虐太久不反弹，弃了」

**启示**：开头可以「土味」「狗血」，但**中段逻辑和付费后爽感**必须兑现。

## 逻辑漏洞自查

| 问题 | 检查 |
|------|------|
| 时间线矛盾 | 第几天发生什么，前后一致？ |
| 身份不一致 | 人名/职位/关系全文统一？ |
| 「为什么不直接 XXX」 | 有没有更简单解法被忽略？ |
| 盟友太轻易 | 帮手是否需要代价/犹豫/拉锯？ |
| 计划太顺 | 是否加入 2-3 次「差点暴露」？ |

## 同质化陷阱

| 过度使用的模式 | 建议 |
|----------------|------|
| 发现物件 → 追查 → 揭秘 → 催泪 | 换结构或换壳 |
| 重生 → 复仇 → 打脸（无新意） | 加独特职业/设定 |
| 追妻火葬场（标准三段） | 换视角或加悬疑线 |

## 投稿路径避坑

### 路径 A：后台投稿

- 创作中心 → 盐选作者 → 创建作品
- 需：梗概 + 完稿
- 审核：约 20 个工作日

### 路径 B：热门问题回答

- 流量更大，但问题调性要匹配
- 虚构需写得「像真的」
- 可同时发专栏 + 问题回答

### 共同避坑

- 不要一稿多投同一审核渠道
- 不要投已饱和的同质题材
- 不要字数卡线（建议 ≥ 12000 字留 buffer）

## 智能体终检清单（拒稿预防）

```
【结构】
□ 主线一句话说清？
□ 全文 ≥ 12000 字？
□ 8-15 章，每章 800-2500 字？
□ 前 4000 字 ≥ 3 钩子？
□ A 级反转在 60-75%？
□ 付费墙在第 3-4 章？

【文风】
□ 第一人称？
□ 段落 1-3 句，独句段 ≥ 30%？
□ 对话 40-50%？
□ 禁用句式清零？
□ 无 4 句以上段落？

【情绪】
□ 有憋屈期 + 强反弹？
□ 主角有能动性？
□ 每 600-800 字有转折？
□ 结尾不升华？

【逻辑】
□ 时间线自洽？
□ 人物前后一致？
□ 无「为什么不直接」漏洞？
□ 有 2+ 次「差点暴露」？

【差异化】
□ 与同类盐选有差异？
□ 未触碰题材红线？
□ 非连续第 3 篇同模式？
```
