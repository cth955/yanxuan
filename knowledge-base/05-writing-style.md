# 05 文风与语言规范

## 文风本质

> **像电影分镜，不像文学作品。**

盐选最佳区间：**有质感的通俗故事** —— 介于纯文学与传统网文之间。

---

## 段落规则（硬性）

| 规则 | 要求 |
|------|------|
| 每段句数 | **1-3 句**，超过必须打散 |
| 独句段占比 | **≥ 30%** |
| 禁止 | **4 句以上的段落** |

### 正误对比

❌ **文学型**（节奏死）：
```
那天下午我回到家，发现门没锁。我心里咯噔一下，推开门走进去。客厅里乱七八糟的，沙发垫子翻了，茶几上的花瓶碎了一地。我站在门口愣了很久，才意识到家里被翻过了。
```

✅ **盐选型**（呼吸感）：
```
那天下午我回到家。
门没锁。
我心里咯噔一下，推开门。
客厅乱七八糟——沙发垫子翻了，茶几上的花瓶碎了一地。
我站在门口愣了很久。
家里被翻过了。
```

---

## 对话规则

| 规则 | 要求 |
|------|------|
| 对话占比 | **40-50%** |
| 格式 | 一句台词一个段落 |
| 标签 | 能省就省 —— 上下文清楚时不需要「他说」 |
| 句长 | 短、毒、不完整 —— 紧张时人不说完整句 |
| 高级技法 | 关键台词复用 —— 同一句在不同场景意义变化 |

### 正误对比

❌：
```
「我觉得你应该告诉我真相，」他严肃地看着我说，「因为欺骗我对你没有任何好处。」
```

✅：
```
「欺骗我，对你没有任何好处。」
```

### 对话进阶

- 允许废话、答非所问、被打断
- **潜台词 > 台词**
- 每个角色必须有独特说话 DNA（用词、句长、口头禅）
- 禁止所有角色都说完整逻辑清晰的长句

---

## 叙述规则

### 叙述者人格

第一人称叙述者必须有**强烈主观性**：

- 吐槽、偏见、情绪失控
- 偶尔跑题 —— 像真人说话时的走神
- 口语化粗口（符合角色身份）
- 连用感叹号、骂人、自我否定（「我他妈居然信了」）

### 感官细节

| 原则 | 说明 |
|------|------|
| 少而精准 | 每场景 **1-2 个**细节就够 |
| 具体 > 抽象 | 「蜂窝煤」>「简陋的厨房」 |
| 禁止 | 铺排景物、大段环境描写 |

### 用词优先级

```
动词 > 形容词 > 副词
```

### 真实感增强

- 具体品牌/地名/APP 名
- 不完美的细节（增加「像真的」）
- 情绪外化 —— 不说「他很愤怒」，写「后槽牙咬得咯吱作响」

---

## 节奏规则

1. **没有任何段落是「过渡」** —— 每段必须推进情节或揭露信息
2. 每 600-800 字一个钩子/转折
3. 开头第一句就要有异常
4. 禁止大段内心独白

---

## 去 AI 味（零容忍）

### 自检三问

1. 匿名发在贴吧，会不会被说「AI 写的」？
2. 遮住名字能不能分辨是谁说话？
3. 段落长度是不是很均匀？（均匀 = AI 特征）

### 禁用句式（叙述文本）

```
❌ 「不是 A，而是 B」/「与其说 A，不如说 B」/「这不仅仅是 A，更是 B」
❌ 「某种意义上来说」/「在这个 A 的时代」/「值得一提的是」
❌ 「不得不说」/「毫无疑问」/「A，而非 B」
❌ 「事实上」作段落开头 / 「然而」作段落开头（全文最多 2 次）
❌ 「心中涌起一股难以言喻的……」/「一种说不清道不明的感觉」
❌ 「XX 如同 XX 一般，XX 着 XX」（隐喻+动名词 AI 句式）
❌ 「不禁……」/「某种程度上」/「那一瞬间」
❌ 「竟然」「居然」「不由自主」（高频出现即改）
```

**唯一例外**：角色对话中该人物说话习惯确实如此（需有人物塑造理由）。

### 人味增强技巧

| 技巧 | 说明 |
|------|------|
| 句式打散 | 连续 5 句节奏相似就打散 |
| 生活闲笔 | 全文至少 5 处（与主线相关的闲笔） |
| 朗读检验 | 手机朗读功能念一遍，别扭处即问题 |
| 动词替换 | 「推开」→「踹开」，增加力度 |
| 不均匀段落 | 故意制造长短交替 |

---

## 盐选 vs 其他风格

| 维度 | 盐选 | 纯文学 | 传统网文 |
|------|------|--------|----------|
| 段落 | 极短，独句段多 | 长短交替 | 中等 |
| 对话 | 40-50% | 视题材 | 30% 左右 |
| 描写 | 一个细节定场景 | 铺陈意境 | 套路描写 |
| 节奏 | 600-800 字一转折 | 慢热可接受 | 章末卡点 |
| 结尾 | 干脆不升华 | 可开放 | 留坑续写 |

---

## 智能体文风自检

```
□ 逐段检查：是否有 4 句以上段落？（有则打散）
□ 独句段是否 ≥ 30%？
□ 对话占比是否 40-50%？
□ 是否有景物/环境铺垫段？（删）
□ 禁用句式是否清零？
□ 每个角色说话能否区分？
□ 是否有 ≥ 5 处生活闲笔？
□ 大声朗读是否顺口？
```
