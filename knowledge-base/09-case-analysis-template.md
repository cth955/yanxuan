# 09 爆款案例拆解模板

## 为什么要拆文

> 外行看热闹，内行看结构。

拆文目的不是模仿剧情，而是提取**可复用的结构、节奏、技法**。

---

## 拆文五步法

```
1. 标记钩子位置 → 2. 提取爽点节奏表 → 3. 画出剧情转折线
   → 4. 分析付费墙 → 5. 提取可复用模板
```

---

## 经典案例类型参考

### 类型 A：生活悬疑

**结构**：日常场景 → 反常发现 → 层层追查 → 真相颠覆 → 主角反击

**示例方向**：
- 「搬进新房的第三天，楼上邻居说他认识我——可我从未见过他」
- 床垫下的身份证、邻居的异常行为

**有效要素**：具体场景 + 第一人称 + 物证驱动

---

### 类型 B：婚姻伦理炸弹

**结构**：亲密关系 → 发现秘密 → 信息层层剥开 → 伦理反转 → 主角抉择

**示例方向**：
- 老公手机里的开房记录
- 私生子的 DNA 鉴定

**有效要素**：数字具体 + 亲属关系 + 道德冲击

---

### 类型 C：重生/古代言情

**结构**：死亡/背叛 → 重生 → 布局 → 逐个打脸 → 终极复仇

**有效要素**：前世记忆信息差 + 读者知道主角知道但反派不知道

---

### 类型 D：追妻火葬场

**结构**：男主忽视/伤害 → 女主离开/不爱了 → 男主追悔 → 女主已有新选择 → 男主难受

**有效要素**：让读者爽在「男的晚了」，不是女主回头

---

### 类型 E：职场逆袭

**结构**：被欺负/被陷害 → 隐忍收集 → 关键节点反击 → 反派反扑 → 绝地翻盘

**有效要素**：具体职场细节 + 扮猪吃虎 + 公开打脸场景

## 拆文练习建议

1. **每周拆 1-2 篇** 高赞/高转化盐选
2. 只拆**结构**，不抄**剧情**
3. 拆完后用模板**反向生成**一个不同题材的大纲
4. 对比自己写的稿与爆款的**节奏表差异**

---

## 智能体拆文模式

当用户提供一篇范文时，智能体应：

1. 按上述模板输出完整拆文
2. 提取 3 个可复用模板
3. 基于模板生成 1 个**不同题材**的原创方案（避免洗稿）

```markdown
## 输出格式
1. 完整拆文（用模板）
2. 可复用模板 × 3
3. 原创方案 × 1（同结构不同故事）
```
