# 01 平台本质与读者画像

## 盐选到底是什么

### 一句话定义

知乎盐选短篇 = **碎片化阅读场景下的情绪快餐**，类似《故事会》《知音》的当代升级版，叠加了算法推荐与会员付费机制。

### 与「知乎精英感」的误区

| 常见误解 | 实际情况 |
|----------|----------|
| 要写深度、洞察、文学性 | 要写的是**强冲突、强情绪、强反转** |
| 文笔越华丽越好 | **节奏和钩子**决定生死，文笔服务于推进 |
| 像写知乎回答一样讲道理 | 像**朋友在讲一件离谱的真事** |

### 平台商业逻辑

```
推荐流曝光 → 标题/导语点击 → 免费区完读 → 付费墙转化 → 会员续费
```

- **标题/导语**：决定约 80% 的打开率
- **前 3000-4000 字**：编辑生死线 + 付费墙位置
- **完读率 > 35%**：才算有推荐潜力；优质稿常追求 60%+

### 「知乎体」的独特性

1. **问答场景嵌入**：故事常伪装成「回答一个问题」或从问题情境切入
2. **第一人称自述**：「我」的视角，增强「像真的发生过」
3. **高密度信息**：每段都要推进，不允许慢热铺垫
4. **短篇幅闭环**：1-2 万字内完成完整情绪弧线

## 读者画像

| 维度 | 数据/特征 |
|------|-----------|
| 年龄 | 20-35 岁为主（约 70%+） |
| 性别 | **女性 90%+**（盐选核心付费人群） |
| 场景 | 地铁、睡前、摸鱼——**手机碎片化** |
| 耐心 | 极低，0.5 秒可划走 |
| 付费意愿 | 女性用户复购率显著高于男性 |

### 盐选读者 vs 番茄读者

| | 盐选读者 | 番茄读者 |
|---|---------|---------|
| 驱动 | **现实共鸣** ——「像不像真的？」 | **纯爽点** ——「刺不刺激？」 |
| 要的是 | 真实感 + 反转快感 + 情绪余韵 | 连续打脸、系统、升级 |
| 不要 | 纯催泪、男频视角、脱离现实脑洞 | 慢热、大段世界观 |

### 读者核心需求（按优先级）

1. ✅ **像真的** —— 具体场景（租房、婚姻、职场、二手房）
2. ✅ **有秘密** —— 发现了不该发现的东西
3. ✅ **有爽感** —— 憋屈后的翻盘/打脸/认知碾压
4. ✅ **有回味** —— 结尾干脆但有余韵，不是纯说教
5. ❌ 纯催泪、纯恐怖、男频龙傲天、脱离生活的玄幻

## 编辑审稿逻辑

编辑不是先评文学性，而是先筛**适配度**：

```
适配度（是不是盐选那味儿）→ 质量（节奏/人物/反转）→ 差异化（同类是否太多）
```

### 官方常见拒稿原因（速记）

1. 主题不合规（敏感、纯色情、宣扬暴力迷信等）
2. **主线不够清晰** —— 一句话说不清讲什么
3. 文笔流畅度不足
4. **风格与盐选相差大** —— 太文学 / 太传统网文 / 太口水
5. **人物不出彩** —— 扁平、无缺陷、无代入
6. **节奏慢** —— 前 4000 字钩子不够
7. **叙事平** —— 无 A 级反转、伏笔弱
8. **同类过多** —— 缺乏差异化
9. 重复投稿

---

## 产品定位总结

> **盐选 ≠ 纯文学**：不追求文字之美，追求故事之力  
> **盐选 ≠ 传统网文**：不要太模板化，要有生活质感  
> **盐选最佳区间**：**有质感的通俗故事** —— 像电影分镜，不像散文

## 给智能体的核心指令

```
你写的不是「小说」，是「情绪产品」。
每一个段落都要么推情节，要么揭露信息，要么制造情绪。
不允许存在「过渡段」「氛围段」「感悟段」。
```
