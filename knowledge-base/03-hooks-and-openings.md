# 03 钩子与开篇

## 开篇生死线

| 区间 | 作用 | 要求 |
|------|------|------|
| **前 10 字（标题）** | 决定会不会点 | 冲突前置，极端事件 |
| **导语（3-5 短句）** | 决定 80% 打开率 | 三有一反 |
| **前 300 字** | 决定会不会划走 | 异常感 + 核心矛盾 |
| **前 500 字** | 第一个正式钩子 | 角色 + 问题 + 为什么特别 |
| **前 2000 字** | 建立关心 | 读者/编辑开始投入 |
| **前 4000 字** | 编辑生死线 | 超级悬念断点 |

---

## 导语设计

### 导语的唯一作用

> 在内容出现在用户眼睛的瞬间，制造**点击冲动**。

### 「三有一反」公式

| 要素 | 说明 | 示例 |
|------|------|------|
| **有内涵** | 直接丢结果，让人想知道前因 | 「婚礼当天，新郎失踪了」 |
| **有悬念** | 留下未解之谜 | 「三天后在冷库被发现」 |
| **有情绪** | 紧张/尴尬/愤怒/害怕 | 「伴郎却穿着他的西装出现」 |
| **反常识** | 违反日常认知 | 「DNA 显示，伴郎是他双胞胎弟弟」 |

### 导语类型

1. **悬念型** —— 抛谜团，不解释
2. **冲突型** —— 直接亮矛盾
3. **反转型** —— 先给结果，再颠覆

### 49 字悬念公式（可选）

```
数字 + 反差 + 命运感
```

**示例**：
```
结婚三年，我在老公手机里发现 100 条开房记录。
收款人，是我亲妹妹。
```

### 导语反模式

❌ 铺垫背景  
❌ 介绍人物履历  
❌ 文艺抒情  
❌ 没有具体事件

## 标题公式

### 四大核心特征

1. **第一人称**「我/她/你」开头 —— TOP100 中 92% 符合
2. **前 10 字就要炸** —— 极端冲突，不要铺垫
3. **具体数字/时间** —— 增强真实感
4. **结尾留悬念** —— 信息缺口

### 五大万能公式

#### 公式 1：身份反差 + 极端事件
```
我（普通身份），却（极端行为/遭遇）
```
> 我是全班最透明的女生，却在毕业典礼上被校长点名上台领奖。

#### 公式 2：亲密关系 + 背叛/秘密
```
和（亲人/爱人）如何如何，直到发现（惊人真相）
```
> 和老公结婚五年，直到我在他保险受益人里看到前女友的名字。

#### 公式 3：时间锚点 + 戏剧反转
```
（具体时间），（反常事件）
```
> 搬进新房的第三天，楼上邻居敲开我的门，说他认识我——可我从未见过他。

#### 公式 4：禁忌选择 + 道德困境
```
为了（目标），我（违背常理的选择）
```
> 为了保住工作，我出卖了最好的闺蜜——直到发现她早就知道。

#### 公式 5：社会热点 + 个人遭遇
```
（热点背景）+（个人极端经历）
```
> 裁员名单公布那天，被裁的是我——而我是写名单的人。

### 标题避雷

| 类型 | 示例 | 问题 |
|------|------|------|
| 文艺抽象 | 《雨夜归人》 | 无冲突，编辑直接拒 |
| 剧透结局 | 《她最终原谅了出轨的丈夫》 | 失去悬念 |
| 过于平淡 | 「我和老公的一次争吵」 | 无极端性 |
| 隐喻象征 | 《时光的裂缝》 | 盐选要求直给冲突 |

## 开篇第一句

### 黄金法则

> **第一句就要有异常** —— 不允许任何铺垫式开头。

### 正误对比

❌ **慢热型**（秒划走）：
```
我是一个普通大学生，直到那天……
```

✅ **爆炸型**：
```
结婚三年，我在老公手机里发现了 100 条开房记录，而对象是我亲妹妹。
```

✅ **悬疑型**：
```
搬进新房的第三天，我在床垫下面摸到了一张不属于我的身份证。
```

✅ **反常型**：
```
老公带回家的私生子，DNA 鉴定出来，是我亲弟弟。
```

### 开篇必须完成的四件事（前 500 字内）

1. **谁** —— 主角身份（一句话）
2. **出了什么事** —— 核心冲突
3. **为什么特别** —— 反常/极端之处
4. **读者为什么要关心** —— 共情点或好奇心

## 钩子类型库

| 类型 | 机制 | 示例 |
|------|------|------|
| 伦理炸弹 | 亲属/爱情 + 禁忌 | 私生子是亲弟弟 |
| 身份错位 | 你以为的 vs 实际 | 老公手机定位在殡仪馆 |
| 物证悬疑 | 具体物件引发追查 | 床垫下的身份证 |
| 数字冲击 | 具体数字增加真实感 | 100 条开房记录 |
| 时间压迫 | 倒计时/期限 | 婚礼前 24 小时 |
| 监控/证据 | 客观记录 vs 主观记忆 | 监控显示我在睡觉——可我明明在凶案现场 |

---

## 一句话验证法则

动笔前，故事概念必须同时满足：

1. **有悬念** —— 听完想知道后面
2. **有反常** —— 违反日常认知
3. **有画面** —— 脑中能成像

三个条件缺一，不值得写。

## 智能体开篇自检

```
□ 标题是否「我/她/你」+ 极端冲突 + 具体细节？
□ 导语是否三有一反？
□ 正文第一句是否有异常感？
□ 前 300 字是否已亮出核心矛盾？
□ 是否禁止了景物/回忆/背景铺垫开头？
□ 读者 0.5 秒内能否感知「这事不对劲」？
```
