# 02 结构与篇幅模板

## 篇幅硬性要求

| 类型 | 字数 | 投稿内容 |
|------|------|----------|
| 短篇故事 | **1万-10万字** | 梗概 + 完稿正文 |
| 实操建议 | **1.2万-2万字** | 过门槛且不显臃肿 |
| 中长篇 | 6万-25万字 | 大纲/梗概 + 3万字试读 |

⚠️ **最低 1 万字**，8000 字不够投稿门槛。

---

## 宏观结构：三幕式 + 五节点

91% 完读率 60%+ 的爆款，常用此框架（3000-5000 字短篇）或等比放大（1-2 万字长篇）。

### 三幕式

| 幕 | 功能 | 关键动作 |
|----|------|----------|
| **第一幕·铺垫** | 前 300-500 字 | 抛钩子 + 主角困境 + 建立共情 |
| **第二幕·对抗** | 主体 60% | 刺激→反击→假胜利→重挫（循环） |
| **第三幕·解决** | 后 20-25% | 终极反转 + 情绪释放 + 干脆收尾 |

### 五节点（第二幕展开）

```
刺激事件 → 首次反击 → 假性胜利 → 重大挫折 → 终极反转
```

**示例**（职场逆袭）：
```
被陷害 → 收集证据 → 以为扳倒了 → 反派反扑 → 绝地反杀
```

---

## 章节结构模板（1.2-2 万字标准稿）

### 基本参数

| 项目 | 要求 |
|------|------|
| 章节数 | **8-15 章** |
| 单章字数 | **1000-2000 字**（不少于 800，不超过 2500） |
| 总字数 | 各章之和 **≥ 13000 字**（留 buffer） |
| 付费墙 | 第 **3-4 章** 结尾（约 3000-4000 字处） |
| A 级反转 | 全文 **60%-75%** 处（约第 8-11 章） |

### 章节功能分配

| 章节 | 功能 | 参考字数 |
|------|------|----------|
| 第 1 章 | 开场钩子 + 角色建立 | 1200-1500 |
| 第 2 章 | 冲突升级 + 世界观补充 | 1200-1500 |
| 第 3 章 | 危机加深 + **付费墙断点** | 1200-1500 |
| 第 4 章 | 付费后第一个爽点/揭秘 | 1000-1500 |
| 第 5-7 章 | 推进 + B/C 级反转 | 每章 1000-1500 |
| 第 8-10 章 | 高潮 + **A 级反转** | 每章 1200-1800 |
| 第 11-12 章 | 收束 + 情绪释放 | 每章 1000-1500 |

---

## 微观结构：反转单元

每个章节 ≈ 一个 **500-800 字的「反转单元」** 叠加而成。

### 「三章必反转」法则

- 每章结尾必设钩子
- 3-5 章内完成一轮起承转合
- 每 2 章左右至少一个「目标 + 阻碍」冲突

**示例**：
```
目标：升职 → 阻碍：被陷害 → 反击：找证据
目标：洗清 → 阻碍：证据被毁 → 反击：换思路
```

---

## 节奏密度要求

| 位置 | 要求 |
|------|------|
| 前 500 字 | 第一个钩子 |
| 前 300 字 | 异常感/核心冲突（更强） |
| 每 600-800 字 | 一个小转折/爽点/信息揭露 |
| 前 4000 字 | 至少 **3 个** 钩子/转折 |
| 付费墙处 | **超级悬念** —— 不可能不往下看 |
| 付费后 | 每 800 字一个情绪高潮，**禁止继续铺垫** |

---

## 反转层级设计

| 级别 | 数量 | 位置 | 作用 |
|------|------|------|------|
| **A 级** | 1 个 | 60%-75% | 颠覆认知，全文最大反转 |
| **B 级** | 1-2 个 | 中后段 | 局部真相揭露 |
| **C 级** | 2-3 个 | 前中段 | 小惊喜、方向修正 |

**优于单一反转**：多层真相结构 > 一次反转

## 大纲格式（智能体输出模板）

```markdown
## 故事一句话
[有悬念 + 有反常 + 有画面]

## 核心冲突
[一句话]

## 主角人设
- 职业：
- 性格缺陷：
- 说话风格示例：

## 情绪过山车
| 章节 | 功能 | 情绪 | 爽点/反转 | 章末钩子 |
|------|------|------|-----------|----------|
| 1 | 钩子 | 惊 | ... | ... |
| ... | ... | ... | ... | ... |

## 付费墙断点
[具体场景描述——断在哪句话]

## A 级反转
[一句话，可剧透]

## 差异化
[与同类盐选的差异点]
```

---

## 梗概要求（投稿必备）

- **300-500 字**
- 包含：核心设定、主角人设、主线、核心反转（可剧透）、结局
- 语言简洁专业，让编辑 10 秒判断商业价值

---

## 结构反模式

❌ 从日常起床/天气/回忆开始  
❌ 大段背景介绍后再进冲突  
❌ 付费墙后还在铺垫  
❌ 结尾升华说教  
❌ 连续 3 章以上无转折  
❌ 所有计划零意外、盟友一问就答应
