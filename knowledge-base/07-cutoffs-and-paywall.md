# 07 卡点与截断技巧

## 付费逻辑

知乎是**会员付费制**。卡点位置决定「转化」—— 读者愿不愿意为作品充值。

```
免费区：下钩子 → 制造疑惑/情绪
付费墙：超级悬念 → 读者忍不住开会员
付费后：第一个爽点/揭秘 → 让读者觉得「值了」
```

> 大反转放在**付费内容里**，免费区只做「勾子」。

---

## 免费区 vs 付费区

| 区间 | 字数 | 核心任务 |
|------|------|----------|
| **免费区** | 前 3000-4000 字（约第 1-3 章） | 钩住 + 建立共情 + 埋伏笔 |
| **付费墙** | 第 3-4 章结尾 | **超级悬念** —— 不可能不往下看 |
| **付费后** | 第 4 章起 | 每 800 字一个高潮，**禁止继续铺垫** |

### 免费区创作重心

> 快节奏、多元素、爆猛料、挑起兴趣。

5000 字免费区要提供足够信息量，但**不足以让读者猜出全故事走向**。

---

## 付费墙设计

### 什么是「超级悬念」

必须是以下之一：

1. **核心秘密揭露前夜** —— 答案呼之欲出但还没给
2. **主角命运重大转折** —— 生死/身份/关系即将颠覆
3. **最大反派即将露面** —— 前面所有线索指向同一人
4. **主角即将做出不可逆选择** —— 读者急想知道选什么

### 正误对比

❌ **弱卡点**（读者可忍）：
```
我决定去质问他。
```
→ 太日常，不构成付费冲动

✅ **强卡点**：
```
我推开那扇门。
他坐在里面，正在签一份文件。
文件的标题是——我的死亡证明。
而我明明还活着。
```
→ 认知颠覆 + 信息缺口 + 生死悬念

## 五种截断形态

> 公式：**情节悬念 + 技术悬念**

| 类型 | 机制 | 示例 |
|------|------|------|
| **转折** | 剧情出现不可控因素 | 「门开了，进来的人不是他，是我死去的父亲」 |
| **情绪** | 铺垫到一半突然中断 | 危机铺垫到顶点 → 戛然而止 |
| **打断** | 行为/对话被突然打断 | 「你到底是——」「别说了！」 |
| **揭秘** | 答案呼之欲出但不给 | 「我终于看清那张脸。是我」——（断） |
| **抉择** | 主角面临二选一 | 「签，还是不签？」 |

### 技术悬念手法

- 段落断在最关键的词上
- 用短句/独句段制造停顿
- 视角突然切换
- 时间跳跃（「三天后」）

## 章末钩子（每章必设）

每一章结尾必须像美剧一样，卡在让人抓心挠肝的地方。

### 章末钩子类型

| 类型 | 适用 |
|------|------|
| 悬念 | 通用 |
| 误会 | 言情 |
| 吃醋 | 言情 |
| 意外肢体接触 | 言情 |
| 新线索 | 悬疑 |
| 身份暗示 | 逆袭/悬疑 |
| 反派露面 | 中后段 |

### 建议做法

每章结尾准备 **3 种可选悬念分支**，选最优路径。

---

## 付费后第一段

> 读者花了钱，要立刻觉得值。

第 4 章（付费后首章）必须：

1. **兑现部分悬念** —— 但不能全揭，留更大悬念
2. **第一个明确爽点** —— 打脸/揭秘/反击
3. **800 字内**有情绪高潮

❌ 付费进来还在铺垫 → 读者骂娘  
✅ 付费进来立刻爆 → 读者觉得值

## 截断公式模板

### 模板 1：身份颠覆

```
[主角行动] + [发现] + [颠覆认知的事实] + [断]
```

### 模板 2：生死悬念

```
[紧张场景] + [关键证据/人物出现] + [与主角认知矛盾] + [断]
```

### 模板 3：抉择悬崖

```
[两难局面铺陈] + [两个选项都可怕] + [主角即将选择] + [断]
```

### 模板 4：关系炸弹

```
[亲密场景] + [无意发现] + [不可接受的真相] + [断]
```

## 卡点常见错误

| 错误 | 后果 |
|------|------|
| 免费区太平 | 无人付费 |
| 卡点太弱 | 转化率低 |
| 付费后继续铺垫 | 读者退款/差评 |
| 卡点剧透太多 | 读者不想付费 |
| 卡点与后文不符 | 信任崩塌（「只有开头好看」） |

---

## 智能体卡点自检

```
□ 免费区（前 4000 字）是否有 ≥ 3 个钩子？
□ 付费墙是否落在第 3-4 章结尾？
□ 付费墙是否是「超级悬念」而非日常推进？
□ 截断是否用了五种形态之一？
□ 付费后第 4 章是否 800 字内有爽点？
□ 每章结尾是否都有向下一章的钩子？
□ 免费区信息是否「不够猜出全故事」？
```
